**UtilsMenu**

Функции
 - Пробив по IP
 - Смс бомбер

**Установка**

Устанавливаем Python и Git

    pkg install python
    pkg install git

Устанавливаем бомбер

    pip install db0mb3r

Устанавливаем UtilsMenu

    git clone https://github.com/artaka/utilsmenu

**Использование**

    cd utilsmenu
    python utilsmenu.py
